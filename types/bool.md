# Bool

The `bool` type has exactly two values: `true` and `false`.

[//]: # (sq types/bool.nut)

```
local y = true;
local n = false;

print("y = " + y + " (" + typeof(y) + ")\n");
print("n = " + n + " (" + typeof(n) + ")\n");
```

```
$ sq types/bool.nut
y = true (bool)
n = false (bool)
```
