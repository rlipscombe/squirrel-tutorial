# Data Types

Squirrel supports the following types:

- [Integer](types/integer.md)
- [Float](types/float.md)
- [String](types/string.md)
- [Null](types/null.md)
- [Bool](types/bool.md)
- [Table](types/table.md)
- [Array](types/array.md)
- Function
- Class
- Class instance
- Generator
- Userdata
- Thread
- Weak references
